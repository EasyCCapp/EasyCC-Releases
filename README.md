# EasyCC Releases

This repository hosts release artifacts and update manifests for EasyCC.

## Update Manifests

- `latest.json` - Stable channel updates
- `latest-beta.json` - Beta channel updates  
- `latest-alpha.json` - Alpha channel updates

## Downloads

Download the latest version of EasyCC from the [Releases](https://github.com/EasyCCapp/EasyCC-Releases/releases) page.

## About EasyCC

EasyCC (Easy Claude Code) is a lightweight desktop app that gives non-technical users the power of Claude Code AI agents without the complexity of IDEs, terminals, or Git.

Learn more at the [EasyCC.app](https://EasyCC.app).
